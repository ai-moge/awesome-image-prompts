# About this library

The prompts in this library are curated mainly from public Twitter/X posts and credited
to their original authors with a source link.

## Submitting a prompt

Have a high-quality image prompt you'd like to see included? Email us at **team@moge.ai**
with the prompt and an example image — great submissions are added to the library with
attribution.

## Removal requests

If you are an author and would like your content removed, email **team@moge.ai** and we
will take it down.
