第 4 格，葡萄乾麵包：
呈現一款真實葡萄乾麵包處於動態未完成狀態，螺旋部分開啟或微微散開，露出亮澤糖霜、葡萄乾團、層次酥點環與黏而酥脆的邊緣。讓幾顆葡萄乾、糖結晶與細碎屑以雕塑般的優雅漂浮於周圍。酥點應感覺亮澤、捲繞、層疊且富節奏。將色盤細化為更深的酒紅葡萄乾、暖焦糖、低飽和梅子與酥點金，賦予更奢華的珠寶般質感。

透過讓四套色彩系統更清晰分離、更聰明地對比，來推進同骨架的細化：

巧克力麵包 = 深可可世界，
杏仁可頌 = 淡奶香杏仁世界，
葡萄乾麵包 = 濃郁酒紅葡萄乾世界。
