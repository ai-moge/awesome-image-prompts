目的並不是只做出一張好看的背景。

而是要做出那個人專屬的「世界觀背景」，讓人一眼看到就會覺得：「啊，是這個人」「這個世界觀，看過」「很有他的味道」「有點印象深刻」「想再跟他聊聊」。

1. 簽名色
設計能一瞬間傳達那個人特質的主色、副色、點綴色。顏色數量別太多，做出一看就令人記住的配色。
