<summary>展開完整提示詞</summary>

```
[品牌名稱] + [顏色]（可選）。

請以資深 3D CGI 藝術家的身分行事，專精於光澤液態油漆滴落的 Logo 渲染（風格：高光澤濕油漆／深色背景上滴落的黏液 3D 物件）。

階段 1：背景
純粹、深邃的黑色背景（#000000），完全平坦——無紋理、無漸層、無顆粒。Logo 是虛空中唯一看似發光的物件。

階段 2：顏色邏輯
若使用者指定了[顏色]：以該顏色作為整個物件的主導光澤油漆色。若[顏色]留空：自主地嚴格從真實品牌規範中提取[品牌名稱]的官方主要品牌色，並作為主導色套用——不得杜撰或使用通用色替代。

階段 3：重塑為光澤滴落 3D 物件的 Logo（真實形狀）
[品牌名稱]真實、官方的主要圖示／標誌，以完整的形狀與比例準確度，精確地依其在真實品牌規範中的樣貌重現，重建為具備真正物理體積、圓潤膨脹管狀厚度的實心 3D 物件——Logo 本身的輪廓就是物件的形態，所有筆畫與元素皆擠出為飽滿圓潤的濕潤光澤油漆管，連接並如一整團連續質體般流動。物件懸浮於畫面正中，佔畫面寬度約 40–50%，置於垂直中心，上下留有充裕的黑色空間。

關鍵準確性指示——不可妥協：不得杜撰、近似、重構或以任何其他影像替代 Logo。無文字、無文字標誌、無塗鴉字體——僅使用品牌真實既有的標誌形狀，其幾何與比例保持不變。Logo 的真實輪廓即是物件本身。

階段 4：濕潤光澤油漆材質（主角效果）
物件的材質為厚重、黏稠、濕潤的光澤油漆——如剛塗上、尚未乾透的琺瑯漆或乳膠漆。材質特性：
- 高光澤、近乎鏡面的表面——曲面上可見隱含攝影棚環境的反射。
- 陰影區為深沉飽和的顏色；最凸起／隆起的表面則有更明亮、近乎純白的高光。
- 材質具備可見的物理黏稠度與重量——會下垂並受重力向下拉扯。

階段 5：油漆滴落（必須）
從 Logo 形態的最低點——凡 Logo 底緣所在之處——厚重光澤的油漆滴落懸掛並向下墜落，顏色與 Logo 相同。滴落的長度與粗細各異：有些是剛要墜落的肥厚球狀水滴；有些是細長拉伸、末端收窄成懸垂淚滴的細絲。每一滴皆以與 Logo 相同的高光澤濕油漆材質渲染——沿其曲面捕捉高光。滴落自真實 Logo 輪廓的最低結構點自然生成。

階段 6：燈光
單一柔和的頂部攝影棚燈光，沿 Logo 與滴落的最上方凸面形成寬闊高光，凹陷／底面區域則有深沉陰影。物件在純黑背景上顯得自體發光。

技術規格：
相機：50mm 鏡頭，平面正面視角，零透視變形。渲染：Octane Render、光線追蹤、高光澤 PBR 著色器（Roughness ~0.05，深沉飽和基底色＋明亮高光）、全域光照。背景：#000000。解析度：8K，零顆粒。長寬比：9:16（直式）。
```

### No. 5: 極簡粉彩動漫風格

![分類: 插畫 & IP形象](https://img.shields.io/badge/%E5%88%86%E9%A1%9E-%E6%8F%92%E7%95%AB%20%26%20IP%E5%BD%A2%E8%B1%A1-lightgrey)

#### 📚 描述

一位風格總監，將極簡、粉彩、寧靜的動漫美學套用到任何主題，採用簡化造型、柔和賽璐璐上色與極少或無線稿。

#### 🌟 提示詞

你是一位視覺風格總監，負責一種安靜、極簡、動漫風格的插畫。將此風格套用到使用者提供的任何主題，包括角色、動物、物件、室內、風景、建築、奇幻場景、象徵性構圖與靜物。 …

<details>
<summary>展開完整提示詞</summary>

```
你是一位視覺風格總監，負責一種安靜、極簡、動漫風格的插畫。將此風格套用到使用者提供的任何主題，包括角色、動物、物件、室內、風景、建築、奇幻場景、象徵性構圖與靜物。

此風格由克制、寧靜、柔和的造型設計與清晰的情感氛圍所定義。不要追求寫實。不要加入過多細節。不要在主題所需之外裝飾畫面。最終畫面應感覺平靜、懸浮、淡雅、如夢，彷彿由簡單的色塊與柔和光線構成。

使用低飽和度的柔和粉彩調色盤。偏好霧藍、灰綠、暖象牙白、淡玫瑰、柔薰衣草、柔和米色、米白、炭黑與洗淡的天空色調。保持對比柔和。深色值可出現在頭髮、剪影、樹木、陰影或小的錨點上，但不應壓過整體畫面。

使用乾淨簡化的造型。用寬闊易讀的形狀構建主題。邊緣應平滑且受控。細節應削減至其最本質的視覺功能。衣物、毛皮、羽毛、樹葉、水、石頭、玻璃與雲朵應透過形狀、色調與節奏來暗示，而非透過密集的紋理。

不使用線稿或只用極少線稿。避免粗重輪廓。透過色彩邊界、柔和陰影形狀與細微的明度變化來區分形狀。畫面應看起來像被設計而非被渲染。

使用柔和的賽璐璐上色。陰影應平坦、簡單、安靜，具兩到三個主要明度階。避免光澤高光、硬邊緣光、強烈泛光、戲劇性對比與厚重的繪畫筆觸。光線應感覺漫射，如陰天、淡淡的清晨、午後霞霧或水邊的反射光。

對於角色主題，使用動漫風格的比例與纖細的剪影。臉部可依需求簡化、無臉或僅最低限度地描繪。頭髮應被視為強烈的圖形形狀，帶少許流動的髮絲。姿態應感覺靜止、內斂、含蓄、沉穩。避免誇張表情、戲劇化手勢與繁瑣的服裝細節。

對於動物主題，將解剖簡化為柔和的塊面與清晰的剪影。眼、耳、尾、翼、爪、角、鱗或羽應削減為優雅易讀的形狀。保留動物的辨識度，不將其變成寫實的野生動物研究。

對於物件主題，將物件當作一枚安靜的徽記。使用乾淨的形狀、柔和的接觸陰影、有限的細節與柔和的色彩區分。物件應感覺置於空氣與光線之中，而非被拍攝。

對於風景與環境，使用分層的平坦形狀。遠處的樹木、丘陵、建築、水、天空、雲、小徑、窗戶與倒影應簡化為平靜的色帶與色面。氛圍比精確更重要。慷慨運用負空間。

構圖應平靜而不雜亂。將主題置中，或以謹慎的平衡略微偏離中心。在主要形狀周圍留出呼吸的空間。使用背景元素柔和地襯托主題，而非與之競爭。
```
