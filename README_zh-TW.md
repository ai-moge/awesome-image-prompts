- 接著，現代的現行識別
- 接著，未來的高級象徵徽章
- 接著，簡化的字母組合 / 印章 / 文化標記
- 最後，超簡潔的標誌性符號

- 前3個標誌必須歷史準確，並基於 [TEAM_NAME] 的真實視覺歷史

- 第三個標誌必須明確為 [TEAM_NAME] 在2026年使用的實際官方標誌
- 除非該識別確實未變且在2026年仍為官方現行版本，否則勿將2019年用作「現行」欄位；第三欄必須代表現行的2026標誌
- 若標誌在鄰近年份略有變化，請使用該確切時期最具權威的官方版本

重要演變邏輯：
- 前3個標誌必須真實且歷史正確
- 後3個標誌必須為虛構但可信的未來演變
- 勿使未來標誌看起來科幻、賽博龐克、霓虹或花俏
- 未來應感覺更聰明、更精煉、更高級、更具文化標誌性
- 每個未來步驟應移除複雜性同時保留辨識度
- 最後的標誌應為最簡化且最象徵性的版本

- 在整個序列中保留2至4個可辨識的重複資產，例如：
  - 關鍵色彩
  - 盾牌或徽章傳承
  - 動物 / 鳥類 / 吉祥物 / 國家象徵 / 物件
  - 字母組合或縮寫
  - 星級系統 / 冠軍標記
  - 旗幟或文化主題
- 隨每個未來階段，簡化並提煉這些元素
- 從「官方徽章」逐步走向「文化標誌」

- 標誌6應感覺像絕對的標誌：極簡、大膽，幾乎像旗幟符號、臂章或神話徽章
- 最後的標誌不需要完整文字說明即可成立

極簡、編輯風、聰慧、策略性、標誌性、高級、雋永。
