第 3 格，杏仁可颂：

巧克力面包 = 深可可世界，
杏仁可颂 = 淡奶香杏仁世界，
葡萄干面包 = 浓郁酒红葡萄干世界。
四格之间的对比在第一眼须更强烈，同时整张板仍感觉像一个高度精选的广告家族。
