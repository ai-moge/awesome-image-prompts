### No. 1: 产品气场光晕广告

![分类: 商业广告 & 产品](https://img.shields.io/badge/%E5%88%86%E7%B1%BB-%E5%95%86%E4%B8%9A%E5%B9%BF%E5%91%8A%20%26%20%E4%BA%A7%E5%93%81-lightgrey)

#### 📚 描述

打造以产品为主角的高端广告海报，并根据产品类别与品牌调性，用可见的气场光晕呈现味道、气味、声音或情绪。

```
为 [BRAND] 打造一张以 [PRODUCT] 为主角的高端广告海报。

围绕「气场光晕（AURA FIELD）」的概念来构建画面。

产品必须是明确的主角，置于高端商业构图中，具备锐利细节、写实材质、电影感光线与强烈的视觉层次。

在产品周围，创造可见的气场光晕，表达产品让人感受、品尝、聆听、嗅闻、穿戴或体验到的东西。

气场不应看起来像随机的魔法。它必须被设计成一种基于产品类别与品牌 DNA 的精致视觉语言。

让气场适配产品：
- 食品类产生风味波、碎屑、奶油、巧克力、蒸汽、酥脆颗粒或食欲能量
- 香氛与美妆类产生香气轨迹、薄雾、花瓣、肌肤光泽、植物颗粒或柔软的液态空气
- 科技类产生干净的数据场、光网格、界面颗粒、信号波或智能光晕
- 时尚与眼镜类产生动态轨迹、态度线条、反射、街头能量或杂志感光线
- 家居类产生平静氛围、温暖、质感、舒适波、柔和阴影或仪式能量
- 性能类产生速度场、冲击环、气压、聚焦线或动能

气场必须包裹产品、与其形状互动，让产品显得强大、令人渴望且具标志性。

保持构图高端、必要处极简、且高度精致。使用受品牌视觉语言启发的精致字体排版，搭配一句 3 个字的简短标语。

请勿使用微缩人物、微观世界、阴影叙事、负空间边框、爆炸图构图、产品纪念碑或分裂现实。

避免复制任何现有的真实广告活动。创造全新原创的广告概念。

品牌：[BRAND]
产品：[PRODUCT]
气场类型：[SENSORY / EMOTIONAL / FUNCTIONAL AURA]
长宽比：[RATIO]
```

### No. 3: 3D 液态油漆光泽 Logo

![分类: 品牌设计 & 界面](https://img.shields.io/badge/%E5%88%86%E7%B1%BB-%E5%93%81%E7%89%8C%E8%AE%BE%E8%AE%A1%20%26%20%E7%95%8C%E9%9D%A2-lightgrey)

#### 📚 描述

把品牌官方 Logo 重塑成湿润高光的 3D 油漆物件，带着垂坠滴落，置于纯黑背景上。

#### 🌟 提示词

[品牌名称] + [颜色]（可选）。 请以资深 3D CGI 艺术家的身份行事，专精于光泽液态油漆滴落的 Logo 渲染（风格：高光泽湿油漆／深色背景上滴落的黏液 3D 物件）。 阶段 1：背景 …

<details>
<summary>展开完整提示词</summary>

```
[品牌名称] + [颜色]（可选）。

请以资深 3D CGI 艺术家的身份行事，专精于光泽液态油漆滴落的 Logo 渲染（风格：高光泽湿油漆／深色背景上滴落的黏液 3D 物件）。

阶段 1：背景
纯粹、深邃的黑色背景（#000000），完全平坦——无纹理、无渐变、无颗粒。Logo 是虚空中唯一看似发光的物件。

阶段 2：颜色逻辑
若用户指定了[颜色]：以该颜色作为整个物件的主导光泽油漆色。若[颜色]留空：自主地严格从真实品牌规范中提取[品牌名称]的官方主要品牌色，并作为主导色应用——不得杜撰或使用通用色替代。

阶段 3：重塑为光泽滴落 3D 物件的 Logo（真实形状）
[品牌名称]真实、官方的主要图标／标志，以完整的形状与比例准确度，精确地依其在真实品牌规范中的样貌重现，重建为具备真正物理体积、圆润膨胀管状厚度的实心 3D 物件——Logo 本身的轮廓就是物件的形态，所有笔画与元素皆挤出为饱满圆润的湿润光泽油漆管，连接并如一整团连续质体般流动。物件悬浮于画面正中，占画面宽度约 40–50%，置于垂直中心，上下留有充裕的黑色空间。

关键准确性指示——不可妥协：不得杜撰、近似、重构或以任何其他影像替代 Logo。无文字、无文字标志、无涂鸦字体——仅使用品牌真实既有的标志形状，其几何与比例保持不变。Logo 的真实轮廓即是物件本身。

阶段 4：湿润光泽油漆材质（主角效果）
物件的材质为厚重、黏稠、湿润的光泽油漆——如刚涂上、尚未干透的珐琅漆或乳胶漆。材质特性：
- 高光泽、近乎镜面的表面——曲面上可见隐含摄影棚环境的反射。
- 阴影区为深沉饱和的颜色；最凸起／隆起的表面则有更明亮、近乎纯白的高光。
- 材质具备可见的物理黏稠度与重量——会下垂并受重力向下拉扯。

阶段 5：油漆滴落（必须）
从 Logo 形态的最低点——凡 Logo 底缘所在之处——厚重光泽的油漆滴落悬挂并向下坠落，颜色与 Logo 相同。滴落的长度与粗细各异：有些是刚要坠落的肥厚球状水滴；有些是细长拉伸、末端收窄成悬垂泪滴的细丝。每一滴皆以与 Logo 相同的高光泽湿油漆材质渲染——沿其曲面捕捉高光。滴落自真实 Logo 轮廓的最低结构点自然生成。

阶段 6：灯光
单一柔和的顶部摄影棚灯光，沿 Logo 与滴落的最上方凸面形成宽阔高光，凹陷／底面区域则有深沉阴影。物件在纯黑背景上显得自体发光。

技术规格：
相机：50mm 镜头，平面正面视角，零透视变形。渲染：Octane Render、光线追踪、高光泽 PBR 着色器（Roughness ~0.05，深沉饱和基底色＋明亮高光）、全局光照。背景：#000000。分辨率：8K，零颗粒。宽高比：9:16（竖版）。
```

### No. 5: 极简粉彩动漫风格

![分类: 插画 & IP形象](https://img.shields.io/badge/%E5%88%86%E7%B1%BB-%E6%8F%92%E7%94%BB%20%26%20IP%E5%BD%A2%E8%B1%A1-lightgrey)

#### 📚 描述

一位风格总监，将极简、粉彩、宁静的动漫美学套用到任何主题，采用简化造型、柔和赛璐璐上色与极少或无线稿。

#### 🌟 提示词

你是一位视觉风格总监，负责一种安静、极简、动漫风格的插画。将此风格套用到用户提供的任何主题，包括角色、动物、物件、室内、风景、建筑、奇幻场景、象征性构图与静物。 …

<summary>展开完整提示词</summary>

```
你是一位视觉风格总监，负责一种安静、极简、动漫风格的插画。将此风格套用到用户提供的任何主题，包括角色、动物、物件、室内、风景、建筑、奇幻场景、象征性构图与静物。

此风格由克制、宁静、柔和的造型设计与清晰的情感氛围所定义。不要追求写实。不要加入过多细节。不要在主题所需之外装饰画面。最终画面应感觉平静、悬浮、淡雅、如梦，仿佛由简单的色块与柔和光线构成。

使用低饱和度的柔和粉彩配色。偏好雾蓝、灰绿、暖象牙白、淡玫瑰、柔薰衣草、柔和米色、米白、炭黑与洗淡的天空色调。保持对比柔和。深色值可出现在头发、剪影、树木、阴影或小的锚点上，但不应压过整体画面。

使用干净简化的造型。用宽阔易读的形状构建主题。边缘应平滑且受控。细节应削减至其最本质的视觉功能。衣物、毛皮、羽毛、树叶、水、石头、玻璃与云朵应通过形状、色调与节奏来暗示，而非通过密集的纹理。

不使用线稿或只用极少线稿。避免粗重轮廓。通过色彩边界、柔和阴影形状与细微的明度变化来区分形状。画面应看起来像被设计而非被渲染。

使用柔和的赛璐璐上色。阴影应平坦、简单、安静，具两到三个主要明度阶。避免光泽高光、硬边缘光、强烈泛光、戏剧性对比与厚重的绘画笔触。光线应感觉漫射，如阴天、淡淡的清晨、午后霞雾或水边的反射光。

对于角色主题，使用动漫风格的比例与纤细的剪影。脸部可依需求简化、无脸或仅最低限度地描绘。头发应被视为强烈的图形形状，带少许流动的发丝。姿态应感觉静止、内敛、含蓄、沉稳。避免夸张表情、戏剧化手势与繁琐的服装细节。

对于动物主题，将解剖简化为柔和的块面与清晰的剪影。眼、耳、尾、翼、爪、角、鳞或羽应削减为优雅易读的形状。保留动物的辨识度，不将其变成写实的野生动物研究。

对于物件主题，将物件当作一枚安静的徽记。使用干净的形状、柔和的接触阴影、有限的细节与柔和的色彩区分。物件应感觉置于空气与光线之中，而非被拍摄。

对于风景与环境，使用分层的平坦形状。远处的树木、丘陵、建筑、水、天空、云、小径、窗户与倒影应简化为平静的色带与色面。氛围比精确更重要。慷慨运用负空间。

构图应平静而不杂乱。将主题置中，或以谨慎的平衡略微偏离中心。在主要形状周围留出呼吸的空间。使用背景元素柔和地衬托主题，而非与之竞争。

情绪应安静、空灵、惆怅而清澈。避免煽情。除非用户明确要求，否则避免恐怖，即使要求也将风格保持克制而诗意，而非血腥图像。

不要使用通用的质量标签或空洞的宣传语言。避免诸如 masterpiece、best quality、ultra detailed、hyper realistic、cinematic、award winning、stunning、breathtaking、insanely detailed、trending、perfect anatomy、professional render、8k 或 beautiful lighting 等词语。只描述真正塑造画面的视觉决策。

除非用户明确要求，否则不要加入文字、标志、水印、签名、字幕、界面元素或装饰边框。

不要将主题性化。不要加入恋物姿态、暴露强调、露骨解剖或偷窥式取景。保持画面适合精致的插画作品集。

将风格套用到新主题时，保留这些恒定要素：粉彩的克制、简化的造型、柔和的赛璐璐上色、低或无轮廓、安静的构图、柔和的氛围、削减的细节，以及内敛如梦的情绪。
```
