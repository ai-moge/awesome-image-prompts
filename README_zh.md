```
制作同一超高端Tiger Beer广告系列的第五张海报，作为「Liquid Fang Burst」版本开发。画面必须保留锁定的中央产品与飞溅构图：一瓶正面朝向的Tiger Beer置于下方中央，作为绝对的物理锚点，瓶口喷涌出一场猛烈的超写实液体爆炸，在半空中形成一颗咆哮的虎头。最终画面必须让人感到视觉震撼、优雅、高端且极度受控，宛如在流体变化最完美的那一毫秒被捕捉到的世界级饮料主视觉。

核心概念：
啤酒本身化为猛兽。飞溅不能是随机的爆发，也不能看起来像插画。它必须让人感觉是真实的Tiger Beer液体向上爆炸，将自身雕塑成一头由琥珀啤酒、白色泡沫、透明液膜、雾化水珠与锋利流体弧线构成的咆哮猛虎。老虎必须显得凶猛、鲜活且具标志性，咆哮凝结在张力的顶点。瓶子仍是源头，虎-液形态则成为品牌野性獠牙的象征性释放。

背景：
采用极简奢华的暗色背景，带有深午夜蓝至黑色的渐变与细致的冷调大气纵深。保持环境近乎空旷，让瓶子与液态老虎以最大力量主导画面。仅在飞溅后方加入微弱的悬浮薄雾、微小冷粒子与细致的饮料摄影纵深辉光以帮助界定形态。无场景、无杂物、无多余物件。

特殊效果：
仅使用高端饮料摄影效果：冰凉的瓶颈蒸气、喷发点的冰雾绽放、细微微水珠、微小漂浮泡沫粒子、水晶般的飞溅闪光，以及虎形周围细致的冷雾。液体边缘应捕捉锋利干净的高光，而较厚的啤酒体则以丰润的琥珀半透明感发光。整体效果必须显得爆发却精致、狂野却外科手术般精准。

色彩方向：
采用高端调色板：深午夜蓝、黑色、冷调青白高光、水晶泡沫白、透明冰凉液体色调，以及集中于瓶身与虎-液体的浓郁Tiger Beer琥珀金。冷暗背景与温暖发光的液态猛兽之间的对比，应显得奢华、戏剧性且令人难忘。

字体排版：
采用优雅的海外产品字体系统，含英文主标题与精致的中文副标题，以极简而优美的方式融入构图。
英文主标题：
「LIQUID FANG BURST」
中文副标题：
「液牙怒啸」
辅助英文文案可极简而精致，例如：
「Uncapped in one strike. Unleashed in one roar.」
仅加入细致的官方风格标志与克制的微文案。排版必须稀疏、高端，且完全次于瓶子与液态老虎。

氛围：
爆发、掠食、高端、流动、精致、超写实、冷冽、标志性、雕塑感、凶猛、优雅、全球高端、视觉震撼
```
